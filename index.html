Парные теги (контейнеры)

Парные теги состоят из открывающего и закрывающего тегов, между которыми помещается содержимое.

Примеры популярных парных тегов:

<html>текст</html> - корневой элемент HTML-документа <head>текст</head> - секция метаинформации <body>текст</body> - основное содержимое страницы <div>текст</div> - блочный контейнер <p>текст</p> - абзац текста <a>текст</a> - гиперссылка <ul>текст</ul> - маркированный список <ol>текст</ol> - нумерованный список <li>текст</li> - элемент списка <h1>текст</h1> ... <h6></h6> - заголовки <span>текст</span> - строчный контейнер <table>текст</table> - таблица <form>текст</form> - форма

Теги для форматирование текста

HTML предоставляет множество тегов для изменения вида текста. Вот основные из них:

<b> - Жирный текст (визуально)

Пример:

Жирный
<strong> - Важный текст (семантически)
